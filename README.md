## DESCRIPTION OF THE PROJECT
Kumari Arpita's Developer Portfolio is a modern, visually stunning personal website designed to showcase my journey, skills, and projects as a final-year Computer Science student based in Ahmedabad, Gujarat. Built with a strong emphasis on aesthetics and user experience, this portfolio reflects my passion for frontend development and my desire to create beautiful, functional web interfaces.

🎨 Design Philosophy
The portfolio embraces a soft, welcoming color palette combining lavender, peach, and sage tones—colors that represent creativity, warmth, and growth. The design system is carefully crafted with custom CSS variables, ensuring consistency throughout the site. The typography pairs elegant Cormorant Garamond for headings with clean DM Sans for body text, creating a perfect balance between sophistication and readability.

✨ Key Features
1. Immersive Hero Section

Dynamic gradient background with animated floating elements

Interactive badge indicating "Open to Opportunities · Fresher 2026"

Gradient-animated name highlight with subtle color shifting

Floating profile card displaying skills and location

Animated statistics showing my current status (fresher, project count, tech stack)

2. About Section

Split layout with personal introduction and technical skills

Interactive skill cards with animated progress bars

Download CV and view projects CTAs

Detailed background information about my education and location

3. Projects Showcase

Grid layout featuring 6 diverse projects across different technologies:

ShopCart: E-commerce UI built with React and localStorage

WIRE: Real-time chat application with WebSocket support

TaskFlow: Task manager with priority labels and local storage

WeatherNow: Live weather app using OpenWeatherMap API

GradePredict: ML model with Flask deployment

ExpenseTracker: Finance tracker with Chart.js visualizations

Each project card includes tags, description, and links to live demo and GitHub

4. Experience Timeline

Two-column layout separating education and internship/certifications

Visual timeline with markers for each milestone

Details about B.Tech in Computer Science (2022-2026)

Frontend Developer Intern experience at TechSpark Solutions

Meta Frontend Developer Certificate and Python certifications

5. Contact Section

Clean, responsive contact form with validation

Social media buttons for GitHub, LinkedIn, Twitter, Instagram, and Email

Form submission simulation with animated feedback

6. Interactive Elements

Smooth scrolling navigation

Responsive mobile menu with hamburger toggle

Back-to-top button that appears on scroll

Intersection Observer for scroll-triggered animations

Active section highlighting in navigation

Custom scrollbar styling

💻 Technical Implementation
Pure HTML, CSS, and JavaScript—no external frameworks or libraries

CSS Variables for consistent theming and easy maintenance

Flexbox and CSS Grid for complex, responsive layouts

Custom animations including fade-up, float, gradient shift, and blink effects

Mobile-first responsive design with breakpoints at 900px and 600px

Backdrop-filter blur for modern glass-morphism effects

Intersection Observer API for performant scroll-based animations

Form validation with simulated submission feedback

🌟 Highlights
